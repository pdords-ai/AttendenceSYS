# Tetris (React + Vite)

## 실행

```bash
npm install
npm run dev
```

프론트엔드: http://localhost:5173
백엔드(API): http://localhost:3000

## 빌드

```bash
npm run build
npm run preview
```

## 조작법
- ←/→: 이동
- ↓: 소프트 드롭
- Space: 하드 드롭
- Z / X: 회전
- Shift: 홀드
- P: 일시정지
- R: 재시작

## 점수 저장/리더보드
- 게임오버 시 이름을 입력하면 점수가 `/api/scores`로 저장됩니다.
- 상위 10개 점수가 리더보드로 표시됩니다.
